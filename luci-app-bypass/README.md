## 特色 

Smartdns 支持

MOSDNS 支持

DNS over Https(DoH) 支持

基于ChinaDNS-NG的国外智能DNS解析, 可与AdGuardHome搭配使用 (AdGuardHome->DNSMASQ->ChinaDNS-NG->SmartDNS)

走代理网站屏蔽IPV6,不影响其他正常网站使用IPV6

国内DNS方案集成

故障转移支持中转节点

功能与视觉体验增强

其他大量修复与优化
